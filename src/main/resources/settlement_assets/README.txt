Settlement Map Assets

Place PNG assets in this folder using the filenames below. Recommended:
- Transparent PNGs (RGBA) with hand-drawn ink + watercolor style.
- Source size around 800x800 to 1600x1600; code will scale down.
- Keep filenames ASCII and exact.

Core map assets
- background_parchment.png (optional; use to replace gradient)
- border_frame.png (hand-drawn border)
- compass_rose.png
- north_arrow.png (optional if not using compass rose)
- scale_bar.png (optional)
- grid_overlay.png (optional hand-drawn grid)
